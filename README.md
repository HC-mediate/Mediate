# 문제인식
<details>

<div markdown="1">
  
## 기존 튜터링 플랫폼 서비스들의 문제점
  
  - 튜터가 클래스를 개설해 수강생들을 모집하지만, 튜터가 튜티를 선택할 수 있는 방법이 없어 접근성이 아쉬움
  - 지역 기반으로 모집하기 때문에 이동거리가 멀면 불편함
  - 튜터링 매칭 후에 관리를 해주는 기능이 미흡함
</div>
</details>
